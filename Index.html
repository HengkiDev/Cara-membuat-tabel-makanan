Cara Membuat Tabel Makanan Favorit dengan HTML

Ingin membuat website sederhana yang menampilkan daftar makanan favorit dengan gambar dan harga? Ikuti panduan ini! Dibuat mudah untuk pemula.

Langkah 1: Buat Struktur Dasar HTML

Mulai dengan membuat file HTML. Simpan dengan nama, misalnya tabel_makanan.html. Tulis kode dasar ini:

<html>
<head>
    <title>Tabel Makanan Favorit</title>
</head>
<body>
    <!-- Tabel akan ditulis di sini -->
</body>
</html>

Penjelasan:

  • <html>: Wadah utama kode HTML.
  • <head>: Berisi judul halaman (<title>).
  • <body>: Tempat konten seperti tabel ditulis.

Langkah 2: Tambahkan Tabel

Gunakan tag <table> untuk membuat tabel. Tambahkan baris (<tr>), judul kolom (<th>), dan data (<td>).

<table>
    <tr>
        <th>Nama Makanan</th>
        <th>Gambar</th>
        <th>Harga</th>
    </tr>
    <tr>
        <td>Nasi Goreng</td>
        <td><img src="link_gambar" alt="Nasi Goreng"></td>
        <td>Rp 25.000</td>
    </tr>
</table>

Penjelasan:

  • <table>: Membuat tabel.
  • <tr>: Membuat baris.
  • <th>: Judul kolom (tebal).
  • <td>: Data dalam kolom.
  • <img>: Menampilkan gambar dengan link (src) dan deskripsi (alt).

Langkah 3: Tambahkan Gambar

Gunakan tag <img> dengan link gambar. Contoh link placeholder: https://via.placeholder.com/100?text=Nasi+Goreng. Jika punya gambar sendiri, simpan di folder yang sama dan tulis nama file, misalnya nasigoreng.jpg.

Langkah 4: Tambahkan Style dengan CSS

Agar tabel terlihat rapi, tambahkan CSS di dalam <head>:

<style>
    table {
        width: 100%;
        border-collapse: collapse;
    }
    th, td {
        border: 1px solid #ddd;
        padding: 8px;
        text-align: left;
    }
    th {
        background-color: #f2f2f2;
    }
    img {
        width: 100px;
    }
</style>

Penjelasan:

  • width: 100%: Tabel selebar halaman.
  • border-collapse: collapse: Menyatukan garis tabel.
  • border: 1px solid #ddd: Garis tepi tabel.
  • padding: 8px: Jarak dalam sel.
  • background-color: #f2f2f2: Warna abu-abu untuk judul.
  • img { width: 100px }: Atur lebar gambar.

Contoh Hasil Tabel

Nama Makanan Gambar Harga
Nasi Goreng Rp 25.000
Sate Ayam Rp 30.000

Langkah 5: Jalankan Website

  1. Simpan kode dalam file dengan akhiran .html, misalnya tabel_makanan.html.
  2. Buka file di browser (Chrome, Firefox, dll.) dengan double-click.
  3. Lihat tabel dengan nama makanan, gambar, dan harga.

Tips untuk Pemula:

  • Ganti teks, harga, atau link gambar sesuai keinginan.
  • Tambah baris (<tr>) untuk makanan lain.
  • Jika gambar tidak muncul, periksa link atau letak file gambar.
  • Simpan perubahan dan refresh browser untuk melihat hasil.
